Ecuaciones cuadráticas especiales
Ecuaciones de la forma $(ax+b)(cx+d)=0$

Objetivo

Resolver ecuaciones cuadráticas del tipo $(ax+b)(cx+d)=0$.

Procedimiento

Para resolver ecuaciones cuadráticas de la forma $(ax+b)(cx+d)=0$ se llevan a cabo los siguientes pasos:

  1. Se iguala a cero cada uno de los dos factores para obtener dos ecuaciones.
  2. Se despeja $x$ de cada ecuación para obtener las dos soluciones.

Solución

Las soluciones son $x_{1} = -\frac{b}{a} $ y $x_{2} = -\frac{d}{c} $. Estas soluciones se obtienen al seguir el procedimiento antes mencionado como se muestra a continuación:

Este tipo de ecuaciones siempre tiene dos soluciones a excepción del caso $\frac{-b}{a} = \frac{-d}{c}$, en el que las dos soluciones son iguales, por lo que se considera que la ecuación tiene una sola solución.

Ejemplos

Utiliza el recuadro interactivo que se presenta a continuación para repasar el método. Pulsa Continuar para desplegar el siguiente paso, y Otro ejemplo para ver más ecuaciones.

Ejercicios

Resuelve en tu cuaderno la ecuación que se muestra en el recuadro interactivo. Después introduce tu respuesta en el campo de texto correspondiente y verifícala. Presiona Otro ejercicio para resolver más ecuaciones. Utiliza tres decimales en las respuestas. Recuerda que puedes realizar operaciones para escribir los resultados.
