Números enteros
Productos con números enteros

Objetivo

Realizar productos con números enteros.

Procedimiento

Se realizan primero las multiplicaciones de signos y posteriormente se resuelven las multiplicaciones con los números.

Solución

Leyes de los signos

$(+)(+) = +$
$(+)(-) = -$
$(-)(+) = -$
$(-)(-) = +$

Al resolver una multiplicación de números enteros, primero se realizan las multiplicaciones de signos empleando las leyes de los signos, que se presentan en el recuadro que se muestra a continuación, y luego se realizan las multiplicaciones de los números.


Ejemplos

Presiona el botón para que aparezcan diferentes ejemplos de multiplicaciones de números enteros.

Ejercicios

Selecciona el número de factores que contendrá el ejercicio y escribe su resultado en el campo de texto del siguiente recuadro interactivo. A continuación presiona el botón Verificar para saber si tu respuesta es correcta. Puedes elegir el número de multiplicaciones o hacer que aparezca un ejercicio nuevo en cualquier momento.
