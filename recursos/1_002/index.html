Números naturales
Operaciones básicas por orden de prioridad

Objetivo

Realizar operaciones combinadas con números naturales considerando el orden de prioridad en las operaciones.

Procedimiento

Para llevar a cabo una operación en la que aparecen sumas, restas, multiplicaciones y divisiones, hay que realizar primero las multiplicaciones y divisiones procediendo de izquierda a derecha y luego las sumas y las restas, en ese mismo orden.

Solución

Cuando en una operación sólo aparecen sumas y restas, la mejor manera de realizarla para no equivocarse es procediendo de izquierda a derecha. En el siguiente recuadro interactivo ensaya este tipo de operaciones, escribe tu respuesta con el botón Resultado y verifícala.

Se puede utilizar el mismo procedimiento para realizar operaciones en las que sólo aparecen multiplicaciones y divisiones. Escribe la respuesta de las operaciones propuestas con el botón Resultado, si es el caso, redondéalo a cuatro decimales y verifica tu respuesta.

Cuando se desea realizar una operación en la que aparecen sumas y/o restas combinadas con multiplicaciones y/o divisiones, la situación es más complicada. Por ejemplo, si para resolver la simple operación $5+8×3$ se realizaran las operaciones de izquierda a derecha, obtendríamos un resultado erróneo.

Las computadoras ejecutan este tipo de operaciones combinadas utilizando una serie de reglas que les permiten decidir el orden en que deben efectuarse las operaciones. Puedes utilizar el siguiente ejercicio para practicar estas reglas. Escribe la respuesta de las operaciones propuestas con el botón Resultado, si es el caso, redondéalo a cuatro decimales y verifica tu respuesta. Luego indica la operación que se llevó a cabo primero para haber obtenido el resultado correcto.

La jerarquía de las operaciones es el conjunto de reglas que permite decidir el orden en que deben realizarse las operaciones. La primera regla es:

1. Se deben realizar primero las operaciones de multiplicación y división y, al final, las de suma y resta.

y la segunda:

2. El orden en que deben ejecutarse las operaciones es de izquierda a derecha.

Estas reglas pueden resumirse en una sola:

Hay que efectuar primero las multiplicaciones y divisiones, operando de izquierda a derecha y luego las sumas y restas, también operando de izquierda a derecha.

Ejercicios

En los siguientes recuadros interactivos deberás utilizar la jerarquía de las operaciones para obtener el resultado correcto. El primer recuadro contiene tres operandos mientras que el segundo es de mayor dificultad pues tiene cuatro.
