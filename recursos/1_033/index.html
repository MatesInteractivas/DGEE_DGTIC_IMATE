Números reales: Orden y operaciones
Operaciones básicas por orden de prioridad

Objetivo

Realizar operaciones combinadas con números reales, considerando el orden de prioridad en las operaciones.

Procedimiento

Cuando se llevan a cabo operaciones con números reales, al igual que con los números racionales, se debe respetar el orden de prioridad en las operaciones, lo cual es una convención universal. Primero se deben realizar todas las multiplicaciones y las divisiones, luego, ya que no haya ninguna multiplicación o división, se realizan las sumas y restas. Dentro de las multiplicaciones y las divisiones, éstas se realizan de izquierda a derecha conforme vayan apareciendo.

Solución

Para realizar operaciones combinadas con números reales, al igual que con los demás conjuntos de números, se debe respetar un orden establecido. Este orden dice que primero se deben realizar todas las multiplicaciones y divisiones de izquierda a derecha y, posteriormente, se realizan las sumas y restas.

Por ejemplo, si se desea resolver $3+2×5$, primero se debe calcular $2×5$ y, posteriormente, el resultado que es igual a $10$ se suma con $3$, lo cual arroja un total de $13$. Si no se respeta este orden y se realiza primero la suma y después el producto se obtendría $254, que es un resultado incorrecto de acuerdo con las convenciones establecidas.

Realiza las operaciones en el orden adecuado y escribe tu resultado en el cuadro de texto.
